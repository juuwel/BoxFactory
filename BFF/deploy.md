### 1. Install Firebase CLI
```bash
npm install -g firebase-tools
```

### 2. Login to Firebase
```bash
firebase login
```

### 3. Make sure dependencies are installed
```bash
npm install
```

### 4. Build the project
```bash
ng build
```

### 5. Initialize Firebase hosting
```bash
firebase init hosting
```

### 6. Deploy to Firebase
```bash
firebase deploy --only hosting
```
